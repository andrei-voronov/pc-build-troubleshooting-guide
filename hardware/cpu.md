# CPU - Central Processing Unit

## Definition
  The CPU is the primary component responsible for executing instructions and perfoming calculations in a computer system.

## Key Features 
  * Cores
  * Threads
  * Clock speed (GHz)
  * Cashe (L1, L2, L3)
  * TDP

## Socket Types
  * LGA 1700
  * AM4
  * AM5

## Common Issues
  * Overheating
  * Bent pins
  * System not POSTing
  * Thermal paste issues

## Troubleshooting
  1. Check cooler
  2. Apply tharmal paste
  3. Verify socket compatibility
  4. Monitor temperatures in BIOS
