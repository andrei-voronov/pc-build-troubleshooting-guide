# RAM – Random Access Memory

## Definition
RAM (Random Access Memory) is volatile memory used to temporarily store data and instructions that the CPU actively processes. All data stored in RAM is lost when power is turned off.

## Types
- DDR3
- DDR4
- DDR5
- DIMM (Desktop modules)
- SO-DIMM (Laptop modules)

## Specifications
- Frequency (MHz)
- Capacity (GB)
- Single Channel / Dual Channel
- ECC vs Non-ECC
- CAS Latency (CL)

## Common Issues
- System not booting
- Memory not detected in BIOS
- Blue Screen errors (BSOD)
- Mixing incompatible modules
- Improper seating of RAM sticks

## Troubleshooting
1. Reseat RAM modules.
2. Test one module at a time.
3. Check BIOS for memory detection.
4. Verify motherboard compatibility.
5. Run memory diagnostic tools (Windows Memory Diagnostic).
