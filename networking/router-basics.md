# Router Basics

## What is a Router?

A router is a network device that forwards traffic between different networks.

It connects:
- Local network (LAN)
- External network (WAN / Internet)

---

## Router Functions

- Routing
- NAT (Network Address Translation)
- DHCP server
- Firewall
- Port forwarding

---

## LAN vs WAN

LAN:
Local Area Network (inside home/office)

WAN:
Wide Area Network (Internet)

---

## NAT

NAT allows multiple private IP addresses to share one public IP address.

Example:
192.168.1.10 → 203.0.113.5

---

## Basic Router Troubleshooting

1. Check WAN connection
2. Verify LAN IP range
3. Restart router
4. Check DHCP status
5. Test connectivity:
   ping 192.168.1.1
   ping 8.8.8.8
